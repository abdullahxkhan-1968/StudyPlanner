# 📘 StudyPlanner

A simple, clean and responsive Study Planner web application to help students manage their daily tasks and study schedule efficiently.

---

## 🌟 Features
- 📝 Add daily study tasks  
- 📅 Organize study schedule  
- 🎯 Easy task tracking  
- 📱 Fully responsive design  
- ⚡ Simple and user-friendly UI  

---

## 🛠️ Tech Stack
-HTML
-CSS
-JavaScript
